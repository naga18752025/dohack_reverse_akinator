🎮 ゲーム内容

1. フクロウが考えたお題（名詞）を当てるゲームです！
（10%の確率で難しめのお題が出ます）

2. フクロウにお題に関する質問をして「はい」「いいえ」「部分的にそう」「たぶんそう」「そうでもない」「たぶん違う」「わからない」という回答を得ることが30回までできます。

3. お題が分かったら『解答する』ボタンから名詞を入力して確定してください。

4. ヒントを使用すると『〜番目の文字は「・・・」です』という情報をゲットできます。

5. リロードは不正操作扱いで強制終了になります。

＊ AIが質問対応をしているため、常に回答が正確であるという保証はできません。
（基本的には正確なはずです）

💡 正解のためのコツ

1. 最初は大まかなカテゴリから特定して絞っていってどんどん狭めていくのが定石です！

2. 質問内容は出来るだけ客観的で答えやすいものにしましょう！
例：❌「それは大きいですか」 ⭕️「それは片手で持てますか」

3. 「みんなの履歴」を見て最近の出題や上手い質問方法、頻出のお題などを確認しておくと後々考える時のヒントになるかも？

4. フクロウは文字数計算に弱いため、ひらがなでの文字数しかわかりません。また、〜字以上（以下）というような質問にも弱いです。

📋 プレイ履歴について

1. 「みんなの履歴」ページでは最近のプレイ順と頻出のお題順でみんなのこれまでの履歴が確認できます！

2. ゲームの正解、ゲーム開始時と質問時の時刻、質問内容とフクロウの回答及びヒントの内容、最終的な解答、プレイ所要時間が履歴として保存されます。

3. 履歴には個人を特定できる情報は一切含まれませんが公開される点をご理解ください。

4. 履歴は内容が不適切であること等の理由で管理者が編集/削除する可能性があります。

👤 アカウントについて

1. 必要なのはユーザー名とパスワードのみで、メールアドレスなどの個人情報は一切不要です！

2.アカウントを作成するとこれまでのプレイ回数と正解数を保存でき、正解数が増えていくとレベルが上がっていきます！
（プレイ履歴とアカウントが紐づいたりはしません）

3. ユーザー名、パスワードは自由に変更することができ、アカウントの削除もいつでもできます。

4. 複数端末でのログインは表示のバグが起きる可能性があるため推奨しません。

5. アカウントのレベルは以下の基準に従って上がっていきます。
